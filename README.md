# Симулятор экономики Казахстана

Интерактивный дашборд: как изменение цен и объёмов сырья проходит сквозь экономику РК —
экспорт/импорт → курс тенге → бюджет → Нацфонд → рост ВВП → инфляция → импорт.

**Живая версия:** https://moldabekovbekbolat30-droid.github.io/kz-econ-sim/

## Состав

| Файл | Назначение |
|------|------------|
| `index.html` | Дашборд (React 18 + Babel через CDN, без сборки). Открывается напрямую в браузере. |
| `kz_model_data.json` | Данные и все коэффициенты модели (выносятся наружу, не зашиты в код). |
| `kz_data_pipeline.py` | Пайплайн обновления данных (НБ РК, World Bank, UN Comtrade, EIA, FRED). |
| `prd.md` | Техническое задание. |

## Запуск локально

```bash
python3 -m http.server 4173
# открыть http://localhost:4173
```

Можно открыть `index.html` и напрямую (`file://`) — есть встроенный фолбэк данных.

## Обновление данных

```bash
# ключи опциональны: без них источник падает в fallback на последнее значение
export COMTRADE_KEY=...   # UN Comtrade
export EIA_KEY=...        # EIA (цена Brent)
python3 kz_data_pipeline.py
```

Падение любого источника не ломает сборку — поле берёт последнее известное значение.

## Модель

Модель статическая, однопериодная; все коэффициенты иллюстративные и собраны в
`kz_model_data.json → coeffs`. При значениях по умолчанию воспроизводит базу 2025–2026
(экспорт $79 млрд, курс 490 ₸/$, рост 5.5%, инфляция 11.0%).
