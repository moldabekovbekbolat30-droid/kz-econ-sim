## 1. Краткое описание

Веб-дашборд, который показывает, как разные факторы — мировые цены сырья, спрос, макрополитика, предложение и внешние условия — проходят по экономике Казахстана: от экспорта и импорта до курса тенге, бюджета, Национального фонда, роста ВВП и инфляции. Пользователь двигает параметры и в реальном времени видит сценарные последствия по всей цепочке, а также **разложение роста ВВП по вкладам** — где сырьё лишь один из драйверов.

Модель связанная: один экран влияет на другой. Данные подтягиваются из официальных открытых источников через отдельный пайплайн.

## 2. Проблема и обоснование

Экономика Казахстана сырьевая: нефть формирует около половины экспорта, а вместе с ураном, медью, ферросплавами и зерном — порядка 60% внешней торговли. Но динамика ВВП объясняется не только ценой сырья, а тем, как она транслируется через бюджетное правило, курс и потребление, и наряду с другими драйверами (инвестиции, монетарная политика, производительность, внешний спрос).

Нужен инструмент, который:
- собирает разрозненные показатели в одну связанную модель;
- показывает причинно-следственные цепочки и **вклад каждого фактора** в рост;
- опирается на актуальные данные и обновляется автоматически.

## 3. Цели и не-цели

### Цели
1. Сценарный инструмент «что если» для цен/объёмов сырья **и** ключевых макродрайверов.
2. Сквозная цепочка: факторы → торговля → курс → бюджет/Нацфонд → ВВП → инфляция → импорт.
3. Разложение роста ВВП по вкладам факторов.
4. Подключить реальные источники данных с автообновлением.
5. Прозрачная и калибруемая модель (коэффициенты вынесены и задокументированы).

### Не-цели (на текущем этапе)
1. Не претендует на точность официальных эконометрических/CGE-прогнозов — модель упрощённая.
2. Не даёт инвестиционных или торговых рекомендаций.
3. Не моделирует временные лаги и краткосрочную волатильность (см. ограничения).
4. Не заменяет данные Минфина/Нацбанка для официальной отчётности.

| Сегмент | Сценарий использования |
|---|---|

## 5. Объём продукта (Scope)

### 5.1 Дашборд — четыре вкладки

**Постоянная панель показателей** (видна всегда): экспорт, импорт, сальдо, курс ₸/$, рост ВВП, инфляция — с дельтой к базе.

**Вкладка «Торговля»**
- Ползунки цен по экспортным группам: нефть, уран, медь, ферросплавы, цинк/свинец, зерно, уголь, газ (50–200%).
- Ползунок автономного спроса на импорт (60–160%).
- Полоса баланса «экспорт vs импорт»; диаграммы структуры экспорта и импорта (база vs сценарий).

**Вкладка «Драйверы роста»** *(новое в 1.1)*
- Совокупный спрос: инвестиции/FDI (60–150%), потребление/доходы (70–140%).
- Макрополитика: базовая ставка НБ РК (8–24%), доля трат сверхдоходов (0–100%).
- Предложение: производительность TFP (85–120%), рабочая сила/занятость (85–115%).
- Внешние факторы: спрос партнёров КНР/РФ/ЕС (70–140%), ставка ФРС (0.5–7%).
- Диаграмма разложения роста ВВП по вкладам (п.п.): базовый уровень → каждый фактор → итог.

**Вкладка «Макроэкономика»**
- Ползунок фискальной политики (дублируется для удобства).
- Карточки: курс, рост ВВП, инфляция, сырьевые доходы бюджета, Δ Нацфонда, сальдо.
- Диаграмма «Бюджет и Нацфонд: база vs сценарий»; пояснение цепочки.

**Вкладка «Источники данных»**
- Таблица: показатель → источник → эндпоинт → формат; дата обновления; инструкция запуска пайплайна.

### 5.2 Пайплайн данных
Скрипт опрашивает источники и собирает единый `kz_model_data.json`, потребляемый дашбордом.

## 7. Спецификация экономической модели

Все коэффициенты иллюстративные и вынесены в код для калибровки. База — наблюдаемое состояние 2025–2026.

### 7.1 Базовые значения

| Параметр | Значение |
|---|---|
| Экспорт (итого) | $79 млрд |
| Импорт (итого) | $65 млрд |
| Торговое сальдо | +$14 млрд |
| Курс USD/KZT | 490 |
| Номинальный ВВП | $290 млрд |
| Рост ВВП | 5.5% |
| Инфляция | 11.0% |
| Базовая ставка НБ РК | 18.0% |
| Ставка ФРС (верх. граница) | 3.75% |
| Активы Нацфонда | $60 млрд |
| Сырьевые доходы бюджета | $22 млрд |
| Базовый трансферт из Нацфонда | $17 млрд |

Драйверы-индексы (база = 100%): инвестиции/FDI, потребление, производительность (TFP), рабочая сила, внешний спрос партнёров.

### 7.2 Расчётные связи

Обозначения: `idx` — индекс цены товара (%/100), `s` — доля трат сверхдоходов, `dr.*` — значения драйверов.

```
Экспорт товара  = base_i × idx_i ;  Экспорт_итого = Σ ;  Сырьевой_экспорт = Σ (без "other")

Курс (экспорт +, ставка ФРС −, ставка НБ РК +):
  USD/KZT = 490 × (Экспорт_итого/79)^(−0.15)
                × (fedRate/3.75)^(0.06)
                × (18/baseRate)^(0.05)

Бюджет и Нацфонд:
  govRev   = 22 + 0.35 × (Сырьевой_экспорт − 61.6)
  windfall = govRev − 22 ;  spent = s × windfall
  transfer = 17 + spent
  nfChange = govRev − transfer + 60 × 0.05

Рост ВВП = базовый уровень (5.5) + Σ вкладов (п.п.):
  сырьё         = 0.04 × (Сырьевой_экспорт − 61.6)
  бюджет        = 0.7 × spent / 290 × 100
  инвестиции    = 0.06 × (invest − 100)
  потребление   = 0.06 × (consume − 100)
  внешний спрос = 0.04 × (extDemand − 100)
  предложение   = 0.05 × (tfp − 100) + 0.04 × (labor − 100)
  ставка НБ РК  = −0.15 × (baseRate − 18)
  ставка ФРС    = −0.10 × (fedRate − 3.75)

Инфляция = 11.0
  + 0.15 × (USD_KZT/490 − 1) × 100        // курсовой перенос
  + 0.5  × spent / 290 × 100              // фискальный спрос
  + 0.04 × (consume − 100) + 0.02 × (invest − 100)   // перегрев спроса
  − 0.12 × (baseRate − 18)                // монетарное сдерживание
  − 0.03 × (tfp − 100)                    // рост производительности

Импорт (эндогенный):
  incomeFactor = 1 + 1.5 × (growth − 5.5)/100
  fxFactor     = (490 / USD_KZT)^0.3
  Импорт_итого = 65 × incomeFactor × fxFactor × (автономный_спрос/100)

Торговое сальдо = Экспорт_итого − Импорт_итого
```

### 7.3 Группы факторов (по каналам влияния)
1. **Совокупный спрос** — потребление, инвестиции/FDI, госрасходы (фискальный импульс), чистый экспорт.
2. **Макрополитика и финусловия** — базовая ставка НБ РК, обменный курс, доступность кредита.
3. **Предложение (потенциал)** — труд, человеческий капитал, производительность (TFP), инфраструктура.
4. **Внешние факторы** — спрос партнёров (Китай/РФ/ЕС), ставка ФРС и глобальные финусловия, геополитика, потоки капитала.

Сырьё входит как экзогенный ценовой шок, вокруг которого выстроены каналы передачи (бюджет, курс, доходы).

## 10. Архитектура и стек

```
[Источники: НБ РК · World Bank · UN Comtrade · EIA · FRED · стат. порталы]
        │  (по расписанию, cron)
        ▼
[kz_data_pipeline.py]  — опрос API, нормализация, fallback
        │
        ▼
[kz_model_data.json]   — статический артефакт данных
        │
        ▼
[Дашборд (React)]      — связанная модель + визуализация (4 вкладки)
```

- **Фронтенд:** React, Recharts; шрифты Fraunces / IBM Plex Sans / Mono; тёмная тема (бирюза + золото).
- **Пайплайн:** Python 3, `requests`, `pandas`, `openpyxl`; cron; ключи через env (`COMTRADE_KEY`, `EIA_KEY`).
- **Поток данных:** сервер/cron собирает JSON; фронтенд читает статический JSON (прямой fetch к API из браузера не используется — CORS).

## 11. Нефункциональные требования

| Категория | Требование |
|---|---|
| Производительность | Пересчёт модели < 50 мс; интерфейс не подвисает при перетаскивании ползунков |
| Надёжность | Падение одного источника не ломает пайплайн (fallback на последние значения) |
| Прозрачность | Все коэффициенты, формулы и источники задокументированы и доступны для правки |
| Поддерживаемость | Товарные группы, драйверы и коэффициенты вынесены в данные/конфиг |
| Доступность | Контраст, читаемые размеры шрифта, подписи на русском |
| Безопасность | Ключи API не хранятся в коде фронтенда; только серверный пайплайн |

## 12. Допущения, ограничения, риски

- Модель статическая и однопериодная — без динамики год-к-году.
- **Лаги политики не учтены:** базовая ставка влияет на инфляцию с задержкой 12–18 мес. (оценка НБ РК), в модели эффект показан как мгновенный.
- Курсовой канал упрощён: реально на тенге влияют рубль, спрос Китая/ЕС, интервенции НБ РК, потоки капитала.
- Не учтены пропускная способность экспорта (КТК), квоты ОПЕК+.
- Часть исторических рядов лежит на `old.stat.gov.kz` — для глубокой истории нужна отдельная обработка.
- У биржи ЕТС нет открытого API; ссылка на xlsx Pink Sheet периодически меняется.

## 13. План развития (Roadmap)

**Фаза 1 (готово)**
- Модель торговли (все товары, экспорт/импорт).
- Макромодель (курс, бюджет, Нацфонд, ВВП, инфляция).
- Драйверы спроса, политики, предложения и внешние факторы; разложение роста ВВП.
- Единый дашборд (4 вкладки) и пайплайн данных из реальных источников.

**Фаза 2**
- Разбивка по странам-партнёрам (Comtrade по partner).
- Многолетний прогноз 3–5 лет с накоплением Нацфонда и трендами цен.
- Калибровка коэффициентов по историческим рядам (регрессия).

**Фаза 3**
- Динамическая версия с временными лагами (особенно монетарная политика).
- Отдельный канал курса рубля и внешнего спроса; ограничения экспорта (КТК, ОПЕК+).
- Сохранение и сравнение сценариев; экспорт результатов.

## 14. Метрики успеха

- Модель воспроизводит базовое состояние 2025–2026 при настройках «по умолчанию».
- Направления реакций экономически корректны (рост цен сырья → крепче тенге; рост ставки → ниже инфляция и рост; и т.д.).
- Разложение роста показывает сопоставимый вклад несырьевых факторов.
- Пайплайн успешно обновляет ≥ 5 из источников при штатном запуске.
- Пользователь за < 1 минуты строит сценарий и читает его последствия.
